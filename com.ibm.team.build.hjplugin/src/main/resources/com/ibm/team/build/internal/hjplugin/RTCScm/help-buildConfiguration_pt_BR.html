Definição de Compilação

A seleção de uma definição de compilação permitirá a integração com a compilação RTC. Isto permitirá que as compilações sejam solicitadas a partir do RTC ou do Hudson/Jenkins. Os resultados de compilação também serão criados em ambos e vinculados um ao outro.

Com a opção de Controle de Versão do Jazz, uma área de trabalho do repositório pode ser especificada juntamente com mais opções de carregamento e aceitação.


Área de Trabalho do Repositório

A seleção de uma área de trabalho do repositório somente aceitará e buscará os conjuntos de mudanças de seu(s) destino(s) de fluxo.

É possível especificar onde a área de trabalho do repositório deve ser carregada, se devem ser criadas pastas de componentes e se o ambiente de simulação deve ser excluído antes do carregamento.


Captura instantânea

A seleção desta opção permite carregar a captura instantânea do Jazz SCM.

É possível especificar onde a área de trabalho do repositório deve ser carregada, se devem ser criadas pastas de componentes e se o ambiente de simulação deve ser excluído antes do carregamento.


Fluxo

A seleção de um fluxo criará uma captura instantânea e fará o carregamento a partir da captura instantânea.

É possível especificar onde a área de trabalho do repositório deve ser carregada, se devem ser criadas pastas de componentes e se o ambiente de simulação deve ser excluído antes do carregamento.
