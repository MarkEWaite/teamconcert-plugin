Havuz çalışma alanına yüklemeyi belirtmek için yükleme kuralları dosyası belirtin. Bir uzantı kullanılarak Dinamik Yükleme Kuralları belirtilirse, burada belirtilenlerden önce gelir.

Yükleme kurallarını uygulamak için aşağıdaki JSON biçiminde bunları listeleyin, listeyi diskte bir dosyaya kaydedin ve bu alanda dosyanın yolunu belirtin. 

  {
    "loadRules":[
      {
        "componentName":"JUnit",
        "filePath":"JUnit Project/junit-project.loadrule"
      },
      {
        "componentName":"Releng",
        "filePath":"Releng Project/releng-project.loadrule"
      }
    ]
  }

"componentName" yerine "componentId" belirtebilirsiniz. "componentId", bileşenin öğe tanıtıcısıdır (UUID). Hem "componentName" hem "componentId" belirtirseniz arama yalnızca "componentId" değerine göre yapılır.

"filePath" değeri, havuzdaki yükleme kuralı dosyasının tam yoludur (uzak yol). "filePath" yerine "fileItemId" belirtebilirsiniz. "fileItemId" yükleme kuralı dosyasının öğe tanıtıcısıdır (UUID). Dosya yolunu belirtirken yol segmentlerini ayırmak için öne eğik çizgileri (/) kullanın. Hem "filePath" hem "fileItemId" belirtirseniz arama yalnızca "fileItemId" değerine göre yapılır.
