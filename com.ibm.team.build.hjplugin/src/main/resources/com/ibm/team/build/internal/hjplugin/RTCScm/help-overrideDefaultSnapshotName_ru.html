Если этот параметр выбран, то можно указать имя моментальной копии, создаваемой после принятия изменений. В противном случае применяется имя моментальной копии по умолчанию. Для заданий на основе определения компоновки по умолчанию применяется имя моментальной копии "${buildDefinitionId}_#${BUILD_NUMBER}". Для заданий на основе потока или рабочей области хранилища по умолчанию применяется имя моментальной копии "${JOB_NAME}_#${BUILD_NUMBER}".
