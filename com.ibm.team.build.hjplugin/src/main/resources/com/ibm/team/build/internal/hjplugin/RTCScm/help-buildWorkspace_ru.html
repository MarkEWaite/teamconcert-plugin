Имя рабочей области хранилища или параметр задания Jenkins, который будет содержать имя рабочей области хранилища на сервере Rational Team Concert (RTC). Параметр задания указывается в формате ${parameter_name}. Должна быть определена только одна рабочая область хранилища с именем. Рабочая область хранилища должна применяться с потоками, которые будут компоноваться.
