A seleção de uma área de trabalho do repositório somente aceitará e buscará os conjuntos de mudanças de seu(s) destino(s) de fluxo.

Caso haja requisitos específicos quanto a como a área de trabalho do repositório deve ser carregada (exemplo: criar pastas de componentes, usar regras de carregamento, excluir o ambiente de simulação antes do carregamento, etc.), use a opção para compilar a partir de uma definição de compilação.
