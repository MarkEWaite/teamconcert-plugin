Vyberte soubor pravidel načtení k určení způsobu načítání pracovního prostoru úložiště. Jestliže byla dynamická pravidla načtení určena pomocí rozšíření, mají tato pravidla přednost před zde určenými.

Chcete-li použít pravidla načtení, vypište je v následujícím formátu JSON, uložte seznam do souboru na disku a určete v tomto poli cestu k souboru. 

  {
    "loadRules":[
      {
        "componentName":"JUnit",
        "filePath":"JUnit Project/junit-project.loadrule"
      },
      {
        "componentName":"Releng",
        "filePath":"Releng Project/releng-project.loadrule"
      }
    ]
  }

Rovněž můžete určit "componentId" namísto "componentName." Hodnota "componentId" je ID položky (UUID) komponenty. Určíte-li obě hodnoty "componentName" a "componentId", dojde k vyhledání pouze podle "componentId."

Hodnota "filePath" je úplná cesta k souboru pravidel načtení v úložišti; např. vzdálená cesta. Rovněž můžete určit "fileItemId" namísto "filePath." Hodnota "fileItemId" je ID položky (UUID) souboru pravidel načtení. Když určíte cestu k souboru, použijte k oddělení segmentů cesty dopředná lomítka (/). Určíte-li obě hodnoty "filePath" a "fileItemId", je vyhledávání provedeno pouze podle "fileItemId."
