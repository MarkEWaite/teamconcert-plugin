Il nome dello spazio di lavoro del repository o un parametro lavoro Jenkins che conterrà il nome dello spazio di lavoro del repository all'interno del server RTC (Rational Team Concert). Specificare il parametro lavoro come ${parameter_name}. È necessario definire un solo spazio di lavoro del repository con il nome. Lo spazio di lavoro del repository deve collaborare con gli stream di cui eseguire la build.
