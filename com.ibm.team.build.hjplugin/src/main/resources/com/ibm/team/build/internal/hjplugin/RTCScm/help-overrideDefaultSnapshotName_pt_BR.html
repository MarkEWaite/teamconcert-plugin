Se essa opção estiver marcada, é possível especificar o nome fornecido à captura instantânea que é criada depois que as mudanças são aceitas. Caso contrário, será usado um nome padrão para a captura instantânea. Para tarefas que usam uma Definição de compilação, o nome da captura instantânea é padronizado como "${buildDefinitionId}_#${BUILD_NUMBER}". Para tarefas que usam um Fluxo ou uma Área de trabalho do repositório, o nome da captura instantânea é padronizado como "${JOB_NAME}_#${BUILD_NUMBER}".
