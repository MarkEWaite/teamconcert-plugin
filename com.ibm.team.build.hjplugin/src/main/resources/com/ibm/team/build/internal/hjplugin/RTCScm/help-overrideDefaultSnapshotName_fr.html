Si cette option est cochée, vous pouvez indiquer le nom donné à l'image instantanée qui est créée après l'acceptation des modifications. Sinon, un nom d'image instantanée par défaut est utilisé. Pour les travaux qui utilisent une définition de génération, le nom d'image instantanée par défaut est "${buildDefinitionId}_#${BUILD_NUMBER}". Pour les travaux qui utilisent un flux ou un espace de travail de référentiel, le nom d'image instantanée par défaut est "${JOB_NAME}_#${BUILD_NUMBER}".
